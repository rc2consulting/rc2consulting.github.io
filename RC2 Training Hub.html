➕ How to Add a New Tool to This Hub
1. Upload the new HTML tool file to your GitHub repository at rc2consulting.github.io
2. Open RC2 Training Hub.html in Text Editor
3. Copy any existing tool card and paste it into the Live Tools section
4. Update the icon, title, description, and filename in the onclick function
5. Save and re-upload to GitHub — the new tool is instantly available to all users
